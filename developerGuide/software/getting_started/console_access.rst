##############
Console Access
##############

Access Red Pitaya's command-line interface through SSH (recommended for most development) or serial console (useful for debugging boot issues).

**Choose your access method:**

* **SSH Access** - Secure remote access over network (most common)
* **Serial Console** - Direct hardware console for boot-level access
