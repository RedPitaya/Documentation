###############
Getting Started
###############

This section covers the basics of accessing and connecting to your Red Pitaya for software development. Before you can develop or deploy software, you need to establish a connection to the Red Pitaya's Linux operating system.

**What you'll find here:**

* **Console Access** - Connect to Red Pitaya via SSH or serial console for command-line access
* **WSL Setup** - Configure Windows Subsystem for Linux for Red Pitaya development (Windows users)
