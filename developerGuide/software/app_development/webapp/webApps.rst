Create your own WEB applications
################################

Develop custom browser-based interfaces for Red Pitaya using the web application framework. These guides cover the application structure, development workflow, and practical examples for creating interactive control panels.

**What you'll find here:**

* **System Overview** - Understand the web application architecture and development environment
* **First Application** - Step-by-step guide to creating your first Red Pitaya web app
* **Examples** - Seven practical examples from LED control to complete measurement interfaces
* **Manual Application Management** - Install and manage web apps without marketplace
