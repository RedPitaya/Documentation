##################################
System Configuration & Integration
##################################

This section covers configuring Red Pitaya's operating system and integrating with external hardware peripherals. Find guides for network setup, GPIO control, communication interfaces, and remote deployment.

**What you'll find here:**

* **Network Configuration** - Set up Wi-Fi, Ethernet, and custom network configurations
* **GPIO Control** - Access and control digital I/O pins from software
* **SPI Interface** - Use the SPI communication interface for peripherals
* **TFT Display** - Connect and configure TFT displays
* **Remote Deployment** - Automate software deployment and updates
