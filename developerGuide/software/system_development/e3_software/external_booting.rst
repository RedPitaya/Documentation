#################################
Booting from external storage
#################################

The Red Pitaya GEN2 boards support booting from external storage devices connected via the E3 connector. Supported external storage options include QSPI flash memory and eMMC modules.

These storage chips should be located on an external module (for example, the :ref:`E3 Add-on board <E3_HW>`).

For custom boot configurations please use AMD (Xilinx) tools and documentation to set up the boot image on the external storage device.

.. note::

    More information regarding booting the Red Pitaya board from external storage will be provided in future documentation updates.
