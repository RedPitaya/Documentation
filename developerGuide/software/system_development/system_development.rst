####################
System Development
####################

This section covers building and modifying the Red Pitaya operating system and ecosystem. These guides are for advanced users who need to compile the OS from source, create custom system images, or develop Debian packages.

**What you'll find here:**

* **Ecosystem Building** - Compile the complete Red Pitaya OS and software ecosystem from source
* **Debian Packages** - Create and build Debian packages for Red Pitaya applications
* **E3 Board Software** - Specific software development for E3 extension boards
