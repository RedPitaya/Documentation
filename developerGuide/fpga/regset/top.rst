.. _fpga_registers:

###########
Registers
###########

This section contains FPGA register maps for different Red Pitaya OS versions. Each OS version may include different FPGA projects 
with varying functionality.

**How to use this section:**

1. Identify your Red Pitaya OS version (check :ref:`System Info <system_info>` or run ``cat /root/.version``)
2. Select the corresponding register map version below
3. Find the specific FPGA project register map you need

**For development:** If you're developing new FPGA projects or using the latest unreleased versions, refer to the :ref:`In Development <regset_in_dev>` section.

For more information about FPGA projects and their features, see :ref:`FPGA Projects <fpga_projects>`.
